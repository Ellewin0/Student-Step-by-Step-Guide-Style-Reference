# Student Step-by-Step Guide Style Reference

This repository contains a public-safe GitHub Pages style reference for creating student-facing step-by-step guides.

## Purpose

Use this page as a public website knowledge source for an agent that formats guides for assistive technology, software, learning tools, and support processes.

The page defines visual style, tone, structure, accessibility rules, reusable components, and agent instructions.

## Important

This style page is public-safe. It does not include internal procedures, private information, organisation-specific policy, support commitments, or contact details.

The page should be used as a style reference only. It should not be used as a source of technical or procedural instructions.

## How to publish with GitHub Pages

1. Create a new public repository.
2. Upload `index.html`, `.nojekyll`, and this `README.md` to the repository root.
3. Go to `Settings` > `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select `main` and `/(root)`.
6. Save.
7. Copy the published GitHub Pages URL and add it as a public website knowledge source.

## Recommended agent settings

- Search all websites: Off
- Only use specified sources: On, after the page is added successfully
- Create documents, charts, and code: On
- Create images: Optional
